## Technologies de panneaux solaires portables

### Silicium monocristallin
- **Rendement** : 18-22 %
- **Avantages** : Haute efficacité, durée de vie longue
- **Inconvénients** : Rigide, coût élevé

### Silicium polycristallin
- **Rendement** : 15-18 %
- **Avantages** : Coût réduit, bonne résistance
- **Inconvénients** : Efficacité moindre

### Silicium amorphe (flexible)
- **Rendement** : 6-10 %
- **Avantages** : Flexible, léger, résistant
- **Inconvénients** : Efficacité faible, dégradation rapide

### Conditions météorologiques
- **Soleil optimal** : 100 % de performance
- **Nuages légers** : 70-80 % de performance
- **Nuages épais** : 30-50 % de performance
- **Pluie** : 10-20 % de performance

## Intégration avec les batteries

### Types de batteries
- **Lithium-ion** : Léger, efficace, coût élevé
- **Lithium-phosphate** : Durable, lourd, coût moyen
- **Plomb-acide** : Économique, lourd, durée de vie limitée

### Gestion intelligente
- Contrôleur de charge MPPT
- Protection contre surcharge
- Indicateurs de charge
- Connexion USB/12V/220V
