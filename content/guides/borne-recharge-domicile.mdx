## Démarches administratives

### 1. Déclaration préalable
- **Assurance habitation** : Déclaration obligatoire
- **Copropriété** : Accord syndic si nécessaire
- **Mairie** : Vérification réglementation locale

### 2. Installation professionnelle
- **Électricien qualifié IRVE** : Obligatoire
- **Certification NFC 15-100** : Mise en conformité
- **Test et mise en service** : Validation installation

## Recommandations par usage

### Usage occasionnel
<Callout type="success">
Prise renforcée 3,7 kW suffisante pour un usage occasionnel.
</Callout>

**Spécifications :**
- Puissance : 3,7 kW
- Installation : Électricien qualifié
- Coût : 1000-1500 €
- Temps de charge : 12-15 h

### Usage quotidien
**Spécifications :**
- Puissance : 7,4 kW
- Installation : Professionnel IRVE
- Coût : 2000-3000 €
- Temps de charge : 6-8 h

### Usage intensif
**Spécifications :**
- Puissance : 11-22 kW
- Installation : Professionnel spécialisé
- Coût : 3000-5000 €
- Temps de charge : 2-4 h
