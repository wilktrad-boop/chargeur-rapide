## Facteurs de choix

### Budget
- **< 50 €** : 30W recommandé
- **50-100 €** : 65W optimal
- **> 100 €** : 100W si nécessaire

### Usage
- **Quotidien smartphone** : 30W suffisant
- **Mix smartphone/ultraportable** : 65W idéal
- **Professionnel intensif** : 100W recommandé

### Mobilité
- **Voyage fréquent** : 30-65W pour compacité
- **Bureau fixe** : 65-100W pour performance
- **Usage nomade** : 30W pour légèreté
