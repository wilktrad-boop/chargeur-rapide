- **Bureau** : Sans fil pour la simplicité
- **Voiture** : Filaire pour la vitesse
- **Nuit** : Sans fil pour l'automatisme
- **Urgence** : Filaire pour la rapidité

- **Écologie** : Filaire pour les économies d'énergie
- **Coût** : Filaire pour le prix d'achat
- **Durée de vie** : Sans fil pour préserver les connecteurs
